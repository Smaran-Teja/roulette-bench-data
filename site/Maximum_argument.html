2 Maximum argument🔗

These benchmarks are produced by the max-arg form, which finds the largest argument for which a computation completes within a timeout. Each plot shows how that maximum argument changes across benchmark runs.

  • max_n-grid (timeout: 1 s, start: 1, end: 200)

  • max_network (timeout: 1 s, start: 1, end: 50)
