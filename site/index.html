Benchmarking results summary🔗

These plots were generated on Wednesday, July 8th, 2026

There are currently 3 parameters on which benchmarks are collected: timing (which includes real time, cpu time and gc time), number of recursive calls, and total size. You can find plots for each parameter in the seperate sections of this document. The plots are generated from the data in the data/ directory.

Scaling benchmarks (those produced by the scale form) are reported separately in the Scaling benchmarks section, with timing, recursive calls and total size shown side-by-side per benchmark; they do not appear in the per-parameter sections above.

Additionally, there are also benchmarks for the code profiler in the Extra plots section. These are generated from the files in the extra directory.

You can find these files and data in the roulette-bench-data repository.

These benchmarks are re-generated with each commit to the roulette repository.
